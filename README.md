# SmartInventory 🚀

Smart Inventory Management System

## Backend
- Node.js + Express
- PostgreSQL database
- JWT authentication
- Email OTP for password reset

## Frontend
- React + Vite + TypeScript
- shadcn/ui components
- Tailwind CSS
- Zustand state management

## Features
- User signup/login/reset
- Dashboard with KPI cards
- Kanban board
- Product management
- Warehouse dashboard
- Adjustments & transfers
- Operations history

## Setup
1. Backend: cd backend && npm install && npm start
2. Frontend: cd inventory-keeper && npm install && npm run dev

## API Endpoints
- POST /api/signup
- POST /api/login  
- POST /api/send-otp
- POST /api/verify-otp
